模組 認證授權
說明 使用指定 QR code 建立新電腦，並回傳電腦基本資料與設定資訊。 此服務用於 device 取得電腦顯示的 QR code 後，使用此 QR code 建立電腦連線用。 調用此服務時，用戶資訊會一併傳送至連線電腦端，建立電腦與 Filelug Cloud 的連線。
回傳值 連線電腦基本資料與設定資訊。
URL POST http://<domain>:<port>/crepo/computer/create-with-qrcode
參數範例（格式：application/json）
{
    "qr-code" : "FOIF8QJO3I48OJFDOFHDUOGBLDSHCNOIFHIEHFUHGDT67GJKBHEIMDOI"
}
參數值說明
名稱 類型 說明
必要參數
qr-code string device 取得電腦顯示的 QR code。
非必要參數
（無）
回傳值範例：（格式：application/json） 
{
    "computer-id" : 9987765,
    "user-id" : "9aaa3acb2bf8aa13533040cc4b24cec22089eba94b4d1c11f3a520656abdab5305b5b3f905b994ea10a24c16783fc340",
    "user-computer-id" : "9aaa3acb2bf8aa13533040cc4b24cec22089eba94b4d1c11f3a520656abdab5305b5b3f905b994ea10a24c16783fc340|9987765",
    "computer-admin-id" : "9aaa3acb2bf8aa13533040cc4b24cec22089eba94b4d1c11f3a520656abdab5305b5b3f905b994ea10a24c16783fc340"
    "computer-group" : "GENERAL",
    "computer-name" : "ALBERT'S WORKSTATION",
    "lug-server-id":"r1",
    "recovery-key":"012336272652"
}
回傳值說明
屬性名稱 類型 說明
必要屬性
user-computer-id string 使用者電腦編號
user-id string 使用者編號
computer-id long 電腦編號
computer-name string 電腦名稱
computer-group string 電腦所屬群組
computer-admin-id string 電腦管理者編號
recovery-key string 要求使用者保管的憑證。要刪除此電腦名稱時需要提供此憑證。為12位數的數字
非必要屬性
lug-server-id string 指定連線的lug server子網域名稱。例如：「r1」表示指定連線的lug server為「r1.filelug.com」。 若回傳值回空值或者lug-server-id不存在，則表示指定連線的lug server為「filelug.com」。
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 session id 未提供，或者必要參數未提供或者參數格式錯誤。
401 session 已過期（預設有效期間為 60 分鐘，自上一次使用此 session id 調用服務的時間開始計算）。
收到此回傳值，前端應調用服務 loginWithSessionId 取回新的 session id 後再重新調用此服務。
403 session 不存在。或者該 session 所對應的用戶不存在。
收到此回傳值，前端應提示用戶使用 Facebook Account Login 重新登入。
465 Device端軟體版本較新，Desktop版本較舊。 Device收到此錯誤碼，應該提示使用者於電腦上下載並安裝新Desktop版本。
466 Device端軟體版本較舊，Desktop版本較新。 Device收到此錯誤碼，應該提示使用者更新Device APP。
500 此服務發生錯誤，請稍後再試。
501 待連線電腦發生下列可能問題，導致連線失敗：
  • QR code 對應不到任何電腦
  • 待連線電腦已經與伺服器失去聯絡。電腦網路不通或者用戶自行關閉電腦上顯示的 QR code 畫面都會造成此情況。

收到此回傳值，前端應提示用戶重新顯示電腦上的 QR code 畫面或者點選目前 QR code 視窗下方按鍵「重新產生二維條碼」，然後使用裝置再次掃描該 QR code。
補充說明
必須先登入系統，取得有效的session id，以作為此服務的 header「fsi」(對 iOS 而言) 或者 header「Authorization」(對非 iOS 而言) 的值。
