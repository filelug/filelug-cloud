Change Computer Name
回上一層
模組 電腦
說明 變更電腦名稱。
回傳值 更新後電腦資訊
URL POST http://<domain>:<port>/crepo/computer/name
參數範例（格式：application/json）
{

    "computer-id" : 3837763683939,
    "new-computer-group" : "GENERAL",
    "new-computer-name" : "ALBERT'S LAPTOP",
    "locale" : "zh_TW"
}
參數值說明
名稱 類型 說明
必要參數
computer-id long 要改名稱的電腦編號
new-computer-name string 電腦新名稱
new-computer-group string 電腦所屬群組
非必要參數
locale string 此設備的 locale
回傳值範例（格式：application/json）
{
    "computer-id" : 3837763683939,
    "computer-group" : "GENERAL",
    "computer-name" : "ALBERT'S LAPTOP"
}
回傳值說明
屬性名稱 類型 說明
必要屬性
computer-id long 要改名稱的電腦編號
computer-name string 變更後的電腦名稱
computer-group string 電腦所屬群組
非必要屬性
（無）
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 必要參數至少一個未提供、提供的 computer group 值不合法、或者對應的電腦資訊不存在。
401 session 已過期（預設有效期間為 60 分鐘，自上一次使用此 session id 調用服務的時間開始計算）。
收到此回傳值，前端應調用服務 loginWithSessionId 取回新的 session id 後再重新調用此服務。
403 session 不存在。或者該 session 所對應的用戶不存在。
收到此回傳值，前端應提示用戶使用 Facebook Account Login 重新登入。
409 新電腦名稱已存在。 遇此錯誤，使用者應該重新選擇新電腦名稱後再執行此服務。
補充說明
必須先登入系統，取得有效的session id，以作為此服務的 header「fsi」(對 iOS 而言) 或者 header「Authorization」(對非 iOS 而言) 的值。
系統提供此服務前，先驗證此 session id 的值是否為有效的 session id， 若session id不存在或者已過期（預設有效期間為60分鐘，自上一次使用此session id調用服務的時間開始計算）， 則回傳錯誤訊息碼401（Unauthorized）。
