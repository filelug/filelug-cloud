Delete Computer From Device
回上一層
模組 電腦
說明 刪除電腦資訊。該電腦不一定為目前用戶 session 中對應的電腦。用戶必須是該電腦的管理員才能刪除此電腦。 若電腦目前連線，則清空該電腦 Filelug 設定並關閉 Filelug 應用程式；若該電腦目前尚未連線，則 下次該電腦 Filelug 啟動時，清空該電腦 Filelug 設定並關閉 Filelug 應用程式。
回傳值 刪除狀態（成功或者失敗）。狀態 200 表示電腦名稱刪除成功。其他訊息詳見下方說明。
URL POST http://<domain>:<port>/crepo/computer/delete2
參數範例（格式：application/json）
{
    "computer-id" : 89765,
    "verification" : "bf94b4d1c5305b5b3f96abdab89eba05b994ea10a249aaa3acb28aa13533040cc4b24cec2783fc3402011f3a52065c16",
    "locale" : "zh_TW"
}
參數值說明
名稱 類型 說明
必要參數
computer-id long 待刪除電腦編號。
verification string 驗證碼。用來確認此服務是由使用者經由授權的device執行。
非必要參數
locale string 前端設備的 locale。若不提供此值，則使用目前 session 內的 locale 值。
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 驗證碼未提供、驗證碼錯誤、或者用戶不是該電腦的管理員。
401 使用者密碼與既有的不相同。
403 使用者帳號尚未註冊。
404 電腦不存在，或者recoveryKey錯誤。
補充說明
必須先登入系統，取得有效的session id，以作為此服務的 header「fsi」(對 iOS 而言) 或者 header「Authorization」(對非 iOS 而言) 的值。
系統提供此服務前，先驗證此 session id 的值是否為有效的 session id， 若session id不存在或者已過期（預設有效期間為60分鐘，自上一次使用此session id調用服務的時間開始計算）， 則回傳錯誤訊息碼401（Unauthorized）。
