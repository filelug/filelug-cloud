findProductsByVendor
回上一層
模組 產品
說明 取得指定廠商的產品資訊
回傳值 指定廠商的產品資訊
URL POST http://<domain>:<port>/crepo/product/findByVendor
參數範例（格式：application/json）
{
  "vendor" : "apple"
}
參數值說明
名稱 類型 說明
必要參數
vendor string 廠商編號
非必要參數
（無）
回傳值範例
[
    {
        "product-id":"com.filelug.apple.product.consumable2g",
        "product-vendor":"apple",
        "vendor-product-type":"Consumable",
        "transfer-bytes" : 2147483648,
        "displayed-transfer-capacity":"2GB",
        "unlimited":false,
        "locale":"zh_TW",
        "product-name" : "2GB 檔案傳輸容量",

]
回傳值說明
屬性名稱 類型 說明
必要屬性
product-id string 產品編號
product-vendor string 廠商編號
vendor-product-type string 廠商對該產品的歸類名稱
transfer-bytes number 新增的傳輸容量。如果傳輸容量無限制，本欄位值為0
displayed-transfer-capacity string 新增的傳輸容量的字串顯示方式。如果傳輸容量無限制，本欄位值為"Unlimited"
unlimited boolean 傳輸容量是否無限制，若是則值為true，否作為false。
locale string 產品資訊顯示語系。
product-name string 產品名稱
product-price number 指定語系下產品價格。
product-displayed-price string 指定語系下產品價格的字串顯示方式。
非必要屬性
（無）
回傳錯誤訊息（HTTP Status）
訊息碼 說明
401 session 已過期（預設有效期間為 60 分鐘，自上一次使用此 session id 調用服務的時間開始計算）。
收到此回傳值，前端應調用服務 loginWithSessionId 取回新的 session id 後再重新調用此服務。
403 session 不存在。或者該 session 所對應的用戶不存在。
收到此回傳值，前端應提示用戶使用 Facebook Account Login 重新登入。
404 未提供vendor資料。
補充說明
必須先登入系統，取得有效的session id，以作為此服務的 header「fsi」(對 iOS 而言) 或者 header「Authorization」(對非 iOS 而言) 的值。
系統提供此服務前，先驗證此 session id 的值是否為有效的 session id， 若session id不存在或者已過期（預設有效期間為60分鐘，自上一次使用此session id調用服務的時間開始計算）， 則回傳錯誤訊息碼401（Unauthorized）。
