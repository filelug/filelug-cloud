回上一層
模組 系統
說明 取得所有支援的電話國碼資訊。依據指定語系顯示國家名稱。
回傳值 所有支援的電話國碼資訊
URL POST http://<domain>:<port>/crepo/system/country
參數範例（格式：application/json）
{
    "ccode-locale" : "zh_TW"
}
參數
名稱 類型 說明
必要參數
locale string 使用者 locale
非必要參數
（無）
