模組 書籤
說明 新增書籤
回傳值 新增的書籤資訊
URL POST http://<domain>:<port>/crepo/bookmarks/new
參數範例（格式：application/json）
{
  "path" : "/Users/albert2013/Documents/mynotes",
  "label" : "Documents",
  "userId" : "TW-0975009123"
}
參數值說明
名稱 類型 說明
必要參數
path string 目錄絕對路徑
label string 目錄名稱。不一定是上一參數「path」的目錄名稱，可以自訂此值
userId string 書籤擁有者編號
非必要參數
（無）
回傳值範例（格式：application/json）
{
    "id":1384740669253,
    "path":"/Users/clopuccino/Google Drive/book",
    "label":"Books",
    "userId" : "TW-0975009123",
    "type":"DIRECTORY" // FILE, WINDOWS_SHORTCUT, or DIRECTORY
}
回傳值說明
屬性名稱 類型 說明
必要屬性
id integer 書籤編號
path string 目錄絕對路徑
label string 目錄名稱
userId string 書籤擁有者編號
type string 書籤的類型。有效值為FILE（書籤路徑為檔案）與DIRECTORY（書籤路徑為目錄）
非必要屬性
（無）
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 可能為下列情況之一：
  • 未指定參數path或指定的path不存在於作業系統
  • 未指定參數label
  • 未指定參數userId
401 session 已過期（預設有效期間為 60 分鐘，自上一次使用此 session id 調用服務的時間開始計算）。
收到此回傳值，前端應調用服務 loginWithSessionId 取回新的 session id 後再重新調用此服務。
403 session 不存在。或者該 session 所對應的用戶不存在。
收到此回傳值，前端應提示用戶使用 Facebook Account Login 重新登入。
409 可能為下列情況之一：
  • 此path已存在於應用程式
  • 此label已存在於應用程式
503 server 與 desktop 間尚未建立連線。
補充說明
  • 必須先登入系統，取得有效的session id，以作為此服務的 header「fsi」(對 iOS 而言) 或者 header「Authorization」(對非 iOS 而言) 的值。
    系統提供此服務前，先驗證此 session id 的值是否為有效的 session id， 若session id不存在或者已過期（預設有效期間為60分鐘，自上一次使用此session id調用服務的時間開始計算）， 則回傳錯誤訊息碼401（Unauthorized）。
  • 當回傳狀態為 200 (OK) 時，回傳的Response Header會包含 field：「Change-Timestamp」， 其值為Desktop端資料異動的時間戳記（單位為millisecond），形態為字串（string），但實際上資料為常整數（long）。
