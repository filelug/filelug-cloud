模組 書籤
說明 新增多筆書籤記錄
回傳值 新增成功的書籤資訊（新增失敗的書簽資訊不回傳）
URL POST http://<domain>:<port>/crepo/bookmarks/synchronize
參數範例（格式：application/json）
[
    {
        "id":1384740669253,
        "path":"/Users/clopuccino/Google Drive/book",
        "label":"Books",
        "userId" : "TW-0975009123",
        "type":"DIRECTORY" // FILE, DIRECTORY, or WINDOWS_SHORTCUT
    },
    {
        "id":1384740669879,
        "path":"/Users/clopuccino/Google Drive/book",
        "label":"Books",
        "userId" : "TW-0975009123",
        "type":"DIRECTORY"
    }
]
參數值說明
名稱 類型 說明
必要參數
id integer 書籤編號
path string 目錄絕對路徑
label string 書籤的名稱。不一定是目錄名稱，可以自訂此值
userId string 書籤擁有者編號
type string 書籤的類型。有效值為FILE（書籤路徑為檔案）、DIRECTORY（書籤路徑為目錄）與WINDOWS_SHORTCUT（書籤路徑為WINDOWS作業系統的捷徑）
非必要參數
（無）
回傳值範例（格式：application/json）

[
    {
        "id":1384740669253,
        "path":"/Users/clopuccino/Google Drive/book",
        "label":"Books",
        "userId" : "TW-0975009123",
        "type":"DIRECTORY" // FILE, DIRECTORY, or WINDOWS_SHORTCUT
    }
    {
        "id":1384740669879,
        "path":"/Users/clopuccino/Google Drive/book",
        "label":"Books",
        "userId" : "TW-0975009123",
        "type":"DIRECTORY"
    }
]
回傳值說明
回傳值包含server新增書簽後，所有的書籤資訊。
屬性名稱 類型 說明
必要屬性
id integer 書籤編號
path string 目錄絕對路徑
label string 書籤的名稱。不一定是目錄名稱，可以自訂此值
userId string 書籤擁有者編號
type string 書籤的類型。有效值為FILE（書籤路徑為檔案）、DIRECTORY（書籤路徑為目錄）與WINDOWS_SHORTCUT（書籤路徑為WINDOWS作業系統的捷徑）
非必要屬性
（無）
回傳錯誤訊息（HTTP Status）
訊息碼 說明
401 session 已過期（預設有效期間為 60 分鐘，自上一次使用此 session id 調用服務的時間開始計算）。
收到此回傳值，前端應調用服務 loginWithSessionId 取回新的 session id 後再重新調用此服務。
403 session 不存在。或者該 session 所對應的用戶不存在。
收到此回傳值，前端應提示用戶使用 Facebook Account Login 重新登入。
503 server 與 desktop 間尚未建立連線。
補充說明
  • 必須先登入系統，取得有效的session id，以作為此服務的 header「fsi」(對 iOS 而言) 或者 header「Authorization」(對非 iOS 而言) 的值。
    系統提供此服務前，先驗證此 session id 的值是否為有效的 session id， 若session id不存在或者已過期（預設有效期間為60分鐘，自上一次使用此session id調用服務的時間開始計算）， 則回傳錯誤訊息碼401（Unauthorized）。
