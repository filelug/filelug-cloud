發送簡訊驗證碼至用戶的電話號碼，以供使用者變更電話號碼
回上一層
模組 認證授權
說明 發送驗證碼至用戶提供的新電話號碼，以供使用者變更電話號碼。
回傳值 無
URL POST http://<domain>:<port>/crepo/user/change-phone-number-code
參數範例（格式：application/json）
{
    "account" : "9aaa3acb2bf8aa13533040cc4b24cec22089eba94b4d1c11f3a520656abdab5305b5b3f905b994ea10a24c16783fc340",
    "passwd" : "81dc9bdb52d04dc20036dbd8313ed055",
    "country-id" : "TW",
    "phone" : "0975009123",
    "verification" : "bf94b4d1c5305b5b3f96abdab89eba05b994ea10a249aaa32065c16",
    "locale" : "zh_TW"
}
參數值說明
名稱 類型 說明
必要參數
account string 使用者編號
passwd string 使用者密碼。此密碼為SHA-256加密後的密碼。
country-id string 新國家編號。
phone string 新電話號碼。依據當地播打電話的規定，可能是「0」開頭。
verification string 系統驗證碼。用來確認此服務是由使用者經由授權的device執行，而不是機器在執行。
locale string 使用者 locale
非必要參數
無
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 必要參數至少一個未提供。或者使用者密碼錯誤。
403 使用者尚未註冊，或者使用者已註冊但是尚未完成驗證。
467 新電話號碼已經有人使用。
500 系統驗證碼錯誤。請求不是來自於授權的應用程式。
補充說明
  • 系統發送使用者驗證碼至新電話號碼，使用者在變更電話號碼畫面輸入收到的使用者驗證碼後， 調用服務「changePhoneNumber」以變更電話號碼。
