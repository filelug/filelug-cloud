incrementBadgeNumber
回上一層
模組 認證授權
說明 增加或減少指定device token的badge number。可一次變更多組。
回傳值 成功變更badge number的device token資訊。
URL POST http://<domain>:<port>/crepo/user/badge-number
參數範例（格式：application/json）
[
    {
        "device-token" : "1e39b345af9b036a2fc1066f2689143746f7d1220c23ff1491619a544a167c61",
        "increment-badge-number" : 1,
        "account" : "9aaa3acb2bf8aa13533040cb5305b5b3f905b994ea10a24c16783fc340"
    },
    {
        "device-token" : "1e39b345af9b036a2fc1066f2689143746f7d1220c23ff1491619a544a167c61",
        "increment-badge-number" : -1,
        "account" : "7bbb0acb2bf8aa13533040cc4b24cec22089eba94b4d1c11f3a520656abdab"
    }
]
參數值說明
名稱 類型 說明
必要參數
device-token string 該device使用的token
increment-badge-number integer 增加（不用額外加「+」）或減少（前面加上「-」號）的badge number。
account string Filelug使用者帳號
非必要參數
無
回傳值範例（格式：application/json）
[
    {
        "device-token" : "1e39b345af9b036a2fc1066f2689143746f7d1220c23ff1491619a544a167c61",
        "notification-type" : "APNS",
        "device-type" : "IOS",
        "device-version" : "8.3",
        "badge-number" : 5,
        "account" : "9aaa3acb2bf8aa13533040cb5305b5b3f905b994ea10a24c16783fc340"
    },
    {
        "device-token" : "1e39b345af9b036a2fc1066f2689143746f7d1220c23ff1491619a544a167c61",
        "notification-type" : "APNS",
        "device-type" : "IOS",
        "device-version" : "8.3",
        "badge-number" : 6,
        "account" : "7bbb0acb2bf8aa13533040cc4b24cec22089eba94b4d1c11f3a520656abdab"
    }
]
回傳值說明
名稱 類型 說明
必要參數
device-token string 該device使用的token
notification-type string Remote Notification Server的類型，有效值為：
  • APNS: For Apple
  • GCM: For Google
  • BAIDU: For Baidu
  • WNS: For Windows 8 (and later) or Windows Phone 8.1 app
  • NONE: Remote notification not supported
device-type string Device的作業系統類型，有效值為：
  • IOS: For apple iOS
  • OSX: For apple Mac OS X
  • WATCH_OS: For apple watchOS
  • WIN_PHONE: For Windows Phone/Pad
  • WIN_DESKTOP: For Windows desktop/server
  • ANDROID: For Android
  • CHROME: For ChromeOS or Chrome browser in desktop/server
  • LINUX_DESKTOP: For Linux desktop/server
  • MOBILE: For mobile devices such as phone and pad, if not applied to the values above
  • DESKTOP: For Desktop/Server, if not applied to the values above
device-version string device作業系統的版號，例如：8.3, 7.1，格式為： 
 ^\\d+(\\.\\d+)+
badge-number integer badge number。此值必為正整數或者0。
account string Filelug使用者帳號
非必要參數
無
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 必要參數不存。詳細資訊顯示於回傳文字訊息。
401 session 已過期（預設有效期間為 60 分鐘，自上一次使用此 session id 調用服務的時間開始計算）。
收到此回傳值，前端應調用服務 loginWithSessionId 取回新的 session id 後再重新調用此服務。
403 session 不存在。或者該 session 所對應的用戶不存在。
收到此回傳值，前端應提示用戶使用 Facebook Account Login 重新登入。
補充說明
必須先登入系統，取得有效的session id，以作為此服務的 header「fsi」(對 iOS 而言) 或者 header「Authorization」(對非 iOS 而言) 的值。
系統提供此服務前，先驗證此 session id 的值是否為有效的 session id， 若session id不存在或者已過期（預設有效期間為60分鐘，自上一次使用此session id調用服務的時間開始計算）， 則回傳錯誤訊息碼401（Unauthorized）。
