模組 認證授權
說明 確認使用者可以刪除，並且沒有允許其他非管理者連接此使用者所屬電腦。
回傳值 該使用者管理的所有電腦名稱，以分行符號分隔。
URL POST http://<domain>:<port>/crepo/user/check-deletable
參數範例（格式：application/json）
{
    "account" : "9aaa3acb2bf8aa13533040cc4b24cec22089eba94b4d1c11f3a520656abdab5305b5b3f905b994ea10a24c16783fc340",
    "locale" : "zh_TW"
}
參數值說明
名稱 類型 說明
必要參數
account string 使用者編號
locale string 使用者 locale
非必要參數
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 必要參數至少一個未提供。
401 未提供使用者session id，或者該session id已經過期。
session id 必須作為 http request header「fsi」或者 「Authorization」值。詳見下方補充說明。
403 可能的情況：
  • 使用者帳號尚未註冊
  • 使用者已註冊，但是尚未驗證
  • 系統不允許刪除此使用者
469 系統允許其他非管理者連接此使用者所屬電腦。
500 系統驗證碼錯誤。請求不是來自於授權的應用程式。
補充說明
必須先登入系統，取得有效的session id，以作為此服務的 header「fsi」(對 iOS 而言) 或者 header「Authorization」(對非 iOS 而言) 的值。
系統提供此服務前，先驗證此 session id 的值是否為有效的 session id， 若session id不存在或者已過期（預設有效期間為60分鐘，自上一次使用此session id調用服務的時間開始計算）， 則回傳錯誤訊息碼401（Unauthorized）。
