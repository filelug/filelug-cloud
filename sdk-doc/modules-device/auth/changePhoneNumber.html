變更電話號碼
回上一層
模組 認證授權
說明 變更電話號碼。
回傳值 無
URL POST http://<domain>:<port>/crepo/user/change-phone-number
參數範例（格式：application/json）
{
    "account" : "9aaa3acb2bf8aa13533040cc4b24cec22089eba94b4d1c11f3a520656abdab5305b5b3f905b994ea10a24c16783fc340",
    "code" : "d04dc2003c5305b5b3f96abdab89eba05b994ea106dbd831",
    "country-id" : "TW",
    "phone" : "0975009123",
    "verification" : "bf94b4d1c5305b5b3f96abdab89eba05b994ea10a249aaa32065c16",
    "locale" : "zh_TW"
}
參數值說明
名稱 類型 說明
必要參數
account string 使用者編號
code string 加密後的使用者驗證碼。此驗證碼由使用者調用「sendChangePhoneNumberSecurityCode」後，系統透過簡訊發送給使用者端。
country-id string 新國家編號。
phone string 新電話號碼。依據當地播打電話的規定，可能是「0」開頭。
verification string 系統驗證碼。用來確認此服務是由使用者經由授權的device執行，而不是機器在執行。
locale string 使用者 locale
非必要參數
無
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 必要參數至少一個未提供。 或者新電話號碼不是之前發送簡訊驗證碼時的新電話號碼。
403 使用者尚未註冊、使用者已註冊但是尚未完成驗證、或者新電話號碼與目前的相同。
431 使用者驗證碼錯誤。請使用者重新輸入。
467 新電話號碼已經有人使用。
500 系統驗證碼錯誤。請求不是來自於授權的應用程式。
補充說明
  • 調用此服務前，必須先調用服務「sendChangePhoneNumberSecurityCode」， 以發送使用者驗證碼至新電話號碼，使用者在變更電話號碼畫面輸入收到的使用者驗證碼後， 再調用此服務以變更電話號碼。
