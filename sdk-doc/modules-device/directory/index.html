名稱 功能 連線伺服器類型
listRoots 取得作業系統所有根目錄路徑 Lug Service
list 列出指定目錄下所有檔案與子目錄 Lug Service
findByPath 列出指定目錄或檔案資訊 Lug Service
findAllFileDownloaded 取得所有已成功下載的檔案資訊 AA Service
findAllFileUploaded 取得所有已成功上傳的檔案資訊 AA Service

下載檔案
名稱 功能 連線伺服器類型
createFileDownloadSummary 建立整批下載摘要資訊（第一版） Lug Service
createFileDownloadSummary2 建立整批下載摘要資訊（第二版） Lug Service
downloadFileToDevice 下載檔案內容 Lug Service
downloadFileToDevice2 下載檔案內容（第二版）。支援檔案續傳，以及整批下載功能。 Lug Service
downloadFileToDevice3 下載檔案內容。支援檔案續傳，以及整批下載功能。對前端調用者而言，此服務與第二版完全相同。 Lug Service
confirmDownloadFileFromDevice 確認檔案下載結果 AA Service
cancelDownloadFileFromDevice 取消下載中的檔案 Lug Service（但不管Socket Connect是否已連線）
replaceFileDownload 整批下載資訊中使用新檔案下載 key 與 file path 取代舊檔案下載 key 與 file path。並刪除 server 中的舊檔案下載資訊。
由於此服務不需要牽動 desktop，因此歸屬於 AA Service。 		AA Service

上傳檔案
名稱 功能 連線伺服器類型
createFileUploadSummary 建立整批上傳摘要資訊（第一版） Lug Service
createFileUploadSummary2 建立整批上傳摘要資訊（第二版） Lug Service
uploadFileFromDevice 上傳檔案。不支援通知功能。此服務已經廢棄，建議改用服務「uploadFileFromDevice2」。 Lug Service
uploadFileFromDevice2 上傳檔案。支援檔案上傳完成後通知，以及整批上傳功能。 Lug Service
findFileUploadedByTransferKey 取得指定檔案上傳資訊。 AA Service
uploadFileFromDevice3 上傳檔案。支援檔案上傳續傳。 Lug Service
uploadFileFromDevice4 上傳檔案。支援檔案上傳續傳。 Lug Service
replaceFileUpload 整批上傳資訊中使用建立新檔案上傳key取代舊檔案上傳key。 並刪除舊檔案上傳資訊，包含Repository與Desktop中的舊檔案上傳資訊都會一併刪除。 Lug Service
confirmUploadFileFromDevice 確認檔案上傳結果。此服務已經廢棄，建議改用服務「confirmUploadFileFromDevice2」。 AA Service
confirmUploadFileFromDevice2 確認檔案上傳結果。支援批次確認。 AA Service
