模組 檔案目錄
說明 取得指定目錄路徑下所有檔案與子目錄
回傳值 指定目錄路徑下所有檔案與子目錄
URL POST http://<domain>:<port>/crepo/directory/list
參數範例（格式：application/json）
{
    "path" : "C:\\Documents and Settings\\Administrator\\My Documents\\My Pictures",
    "showHidden" : false
}
參數值說明
名稱 類型 說明
必要參數
path string 目錄或檔案的絕對路徑。此路徑值不須要encoded。
URL encoding的規則詳見：HTML URL Encoding Reference，其中的「URL Encoding Functions」可用來encode URL
非必要參數
showHidden boolean 是否顯示隱藏檔，若要顯示隱藏檔則設為 true，否則設為 false。
若此值設為 true，會顯示隱藏檔與非隱藏檔；若此值設為 false，則僅顯示非隱藏檔。 若此值與目前 session 中的 showHidden 屬性值不同，則以此為準。
若此值不提供，則使用目前 session 中的 showHidden 值。

回傳值說明
屬性名稱 類型 說明
必要屬性
symlink boolean 是否為symbolic link。對Windows作業系統而言，此值永遠是false。
name string 檔案或子目錄名稱。
parent string 父目錄。亦即參數path值
readable boolean 是否有讀的權限
writable boolean 是否有寫的權限
executable boolean 是否有執行的權限
hidden boolean 是否為隱藏檔
lastModified string 最後修改日期。格式為「yyyy/MM/dd HH:mm:ss 'GMT'Z」
type string 目錄或檔案。目錄值以「DIRECTORY」結尾、檔案值以「FILE」結尾。 有效值包含：
  • FILE,
  • DIRECTORY,
  • WINDOWS_SHORTCUT_FILE,
  • WINDOWS_SHORTCUT_DIRECTORY,
  • UNIX_SYMBOLIC_LINK_FILE,
  • UNIX_SYMBOLIC_LINK_DIRECTORY,
  • MAC_ALIAS_FILE,
  • MAC_ALIAS_DIRECTORY
非必要屬性
contentType string type值以「DIRECTORY」結尾時，此值永遠是「application/directory」。 若type值以「FILE」結尾時，此值為空值。若要取得此值，須調用服務「findByPath」。
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 指定的目錄絕對路徑不存在
401 session 已過期（預設有效期間為 60 分鐘，自上一次使用此 session id 調用服務的時間開始計算）。
收到此回傳值，前端應調用服務 loginWithSessionId 取回新的 session id 後再重新調用此服務。
403 session 不存在。或者該 session 所對應的用戶不存在。
收到此回傳值，前端應提示用戶使用 Facebook Account Login 重新登入。
503 server 與 desktop 間尚未建立連線。
補充說明
  1. 調用此服務，可於header新增「Accept-Encoding: gzip, deflate」幾可要求repository壓縮回傳資料。
  2. 若要取得檔案或目錄的大小（size），須調用服務「findByPath」， 並設定參數「calculateSize」值為「true」。
  3. 必須先登入系統，取得有效的session id，以作為此服務的 header「fsi」(對 iOS 而言) 或者 header「Authorization」(對非 iOS 而言) 的值。
    系統提供此服務前，先驗證此 session id 的值是否為有效的 session id， 若session id不存在或者已過期（預設有效期間為60分鐘，自上一次使用此session id調用服務的時間開始計算）， 則回傳錯誤訊息碼401（Unauthorized）。
