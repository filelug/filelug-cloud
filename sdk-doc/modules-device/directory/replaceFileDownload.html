模組 檔案目錄
說明 整批下載資訊中使用新檔案下載 key 取代舊檔案下載 key。並刪除 server 中的舊檔案下載資訊。
回傳值 刪除指定檔案下載資訊結果。
URL POST http://<domain>:<port>/crepo/directory/replace-download
參數範例（格式：application/json）
{
    "old-transferKey" : "OTFDRjUxNzjglQUMlRTglQTklQTYuUE5HKzEzOTI2MjEwMjQzNDc=1",
    "new-transferKey" : "OTFDRjUxNzjglQUMlRTglQTklQTYuUE5HKzEzOTI2MjEwMjQzNDc=2"
}
參數值說明
名稱 類型 說明
必要參數
old-transferKey string 舊下載檔案key。意同服務createFileDownloadSummary>的參數「download-key-paths」的 key 值。
new-transferKey string 新下載檔案key。意同服務createFileDownloadSummary>的參數「download-key-paths」的 key 值。
非必要參數
（無）
回傳值說明
屬性名稱 類型 說明
必要屬性
（無）
非必要屬性
（無）
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 必要參數為提供，或者指定的 old-transferKey 不存在。
401 session 已過期（預設有效期間為 60 分鐘，自上一次使用此 session id 調用服務的時間開始計算）。
收到此回傳值，前端應調用服務 loginWithSessionId 取回新的 session id 後再重新調用此服務。
403 session 不存在。或者該 session 所對應的用戶不存在。
收到此回傳值，前端應提示用戶使用 Facebook Account Login 重新登入。
補充說明
必須先登入系統，取得有效的session id，以作為此服務的 header「fsi」(對 iOS 而言) 或者 header「Authorization」(對非 iOS 而言) 的值。
系統提供此服務前，先驗證此 session id 的值是否為有效的 session id， 若session id不存在或者已過期（預設有效期間為60分鐘，自上一次使用此session id調用服務的時間開始計算）， 則回傳錯誤訊息碼401（Unauthorized）。
