模組 檔案目錄
URL POST http://<domain>:<port>/crepo/directory/supload2
說明 上傳檔案內容給 server，即 Device 端下載檔案時，Desktop 端使用此服務上傳檔案。 HttpServletRequest物件的 header 中，設定「Authorization」的值為 file transfer key。 當檔案內容「開始傳輸」給前端後，就回傳status 200; 若沒有 file transfer key 或者找不到 file transfer key 對應的資料，則回傳 status=400; 若尚未傳輸給前端時就發生其他錯誤，則回傳 status=500
回傳錯誤訊息（HTTP Status）
訊息碼 說明
401 沒有 file transfer key 或者找不到 file transfer key 對應的資料。
403 無法回傳到前端。
500 尚未傳輸給前端時就發生其他錯誤。
