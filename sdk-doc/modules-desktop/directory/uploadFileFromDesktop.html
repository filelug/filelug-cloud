模組 檔案目錄
URL POST http://<domain>:<port>/crepo/directory/supload
說明 上傳檔案內容。 系統利用client session找到對應的HttpServletResponse後，使用此物件回傳給前端device。 HttpServletRequest物件的header中，設定「Authorization」的值為client session id。 當檔案內容「開始傳輸」給前端後，就回傳status 200; 若沒有client session id或者找不到client session id對應的HttpServletResponse，則回傳 status=400; 若尚未傳輸給前端時就發生其他錯誤，則回傳 status=500
回傳錯誤訊息（HTTP Status）
訊息碼 說明
401 沒有client session id或者找不到client session id對應的HttpServletResponse。
403 無法回傳到前端。
500 尚未傳輸給前端時就發生其他錯誤。
