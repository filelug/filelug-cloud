上傳下載檔案
名稱 功能 連線伺服器類型
downloadFileToDesktop 從 server 下載檔案到 Desktop（Device端上傳檔案時，Desktop端使用此服務下載檔案） Lug Service
downloadFileToDesktop2 從 server 下載檔案到 Desktop（V2）（Device端上傳檔案時，Desktop端使用此服務下載檔案）。此服務支援續傳。 此服務「不適用」於 device 與 desktop 版本皆為 2.0.0 或者以上的版本。 Lug Service
downloadFileToDesktop3 從 server 下載檔案到 Desktop（V3）（Device端上傳檔案時，Desktop端使用此服務下載檔案）。 此服務支援續傳，method 改為 GET，而且必須提供 desktop session id。 此服務只適用於 device 與 desktop 版本皆為 2.0.0 或者以上的版本。 Lug Service
uploadFileFromServer 上傳檔案到 server（Device端下載檔案時，Desktop端使用此服務上傳檔案）。此服務支援續傳。 Lug Service
uploadFileFromServer2 上傳檔案到 server（V2）（Device端下載檔案時，Desktop端使用此服務上傳檔案）。此服務支援續傳。 此服務「不適用」於 device 與 desktop 版本皆為 2.0.0 或者以上的版本。 Lug Service
uploadFileFromServer3 上傳檔案到 server（V3）（Device端下載檔案時，Desktop端使用此服務上傳檔案）。此服務支援續傳，而且必須提供 desktop session id。 此服務只不適用於 device 與 desktop 版本皆為 2.0.0 或者以上的版本。 Lug Service
cancelDownloadFileFromServer 取消執行中的檔案下載 Lug Service
