參數值說明
名稱 類型 說明
必要參數
old-account string 目前管理者帳號。
old-passwd string 目前管理者密碼。此密碼為SHA-256加密後的密碼
new-account string 新管理者帳號。
new-passwd string 新管理者密碼。此密碼為SHA-256加密後的密碼
computer-id long 連線電腦編號
verification string 驗證碼。用來確認此服務是由使用者經由授權的device執行，而不是機器在執行。
非必要參數
（無）
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 使用者帳號與密碼至少一個未提供，或者提供的 computer group 值不合法。
401 可能的原因：
  • 目前管理者密碼錯誤。
  • 新管理者密碼錯誤<。/li>
  • 目前管理者尚未註冊。
  • 新管理者尚未註冊。
403 目前管理者或新管理者不存在。
409 新電腦名稱已存在。 遇此錯誤，使用者應該重新選擇新電腦名稱後再執行此服務。
460 電腦不存在。
461 old-account不是電腦目前管理者。
462 new-account不允許存取此電腦。包含已申請，但是尚未允許。
500 內部錯誤或驗證碼錯誤。
補充說明
無。
