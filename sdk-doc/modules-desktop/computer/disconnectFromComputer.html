Disconnect from Computer
回上一層
模組 電腦
URL POST http://<domain>:<port>/crepo/computer/disconnect
說明 斷開與 server 的 socket 連線。不論該連線是否為電腦管理者，斷線時都使用此服務。
無參數
無回傳值
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 session id 未提供。
403 session 不存在。或者該 session 所對應的用戶不存在。
收到此回傳值，前端提示用戶重新開啟應用程式，重新進行設定。
補充說明
  • 必須先登入系統，取得有效的session id，以作為此服務的 header「fsi」(對 iOS 而言) 或者 header「Authorization」(對非 iOS 而言) 的值。
  • 系統收到請求後，預設會延遲 5 秒後再斷線。若提供的 session id 已經過期，則仍斷線，不會回傳錯誤碼 401。
