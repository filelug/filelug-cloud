Find User By User ID
回上一層
模組 認證授權
說明 使用國家或地區、與電話號碼取得使用者資訊。
回傳值 取得的使用者資訊。若找不到
URL POST http://<domain>:<port>/crepo/user/find-by-phone
參數範例（格式：application/json）
{
    "country-id" : "TW",
    "phone" : "0975009123",
    "passwd" : "81dc9bdb52d04dc20036dbd8313ed055",
    "nickname" : "Jellyfish",
    "verification" : "bf94b4d1c5305b5b3f96abdab89eba05b994ea10a249aaa3acb28aa13533040cc4b24cec2783fc3402011f3a52065c16",
    "locale" : "zh_TW"
}
參數值說明
名稱 類型 說明
必要參數
country-id string 國家編號
phone string 電話號碼。依據當地播打電話的規定，可能是「0」開頭。
passwd string 使用者密碼。此密碼為SHA-256加密後的密碼
nickname string 使用者暱稱
verification string 驗證碼。用來確認此服務是由使用者經由授權的device執行，而不是機器在執行。
非必要參數
（無）
回傳值範例（格式：application/json）
（注意：回傳值不包含使用者密碼資訊）
{
    "account" : "9aaa3acb2bf8aa13533040cc4b24cec22089eba94b4d1c11f3a520656abdab5305b5b3f905b994ea10a24c16783fc340",
    "nickname" : "Jellyfish",
    "showHidden" : false
}
回傳值說明
名稱 類型 說明
必要參數
account string 使用者帳號。
nickname string 使用者暱稱
showHidden boolean 是否顯示隱藏檔。
非必要參數
（無）
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 使用者帳號與密碼至少一個未提供。
401 指定使用者密碼錯誤。
403 找不到指定國家或地區、與電話號碼的使用者，或者使用者已註冊，但是尚未驗證。
補充說明
無
