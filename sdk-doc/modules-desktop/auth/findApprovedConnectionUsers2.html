Find approved connection users of a computer(V2)
回上一層
模組 認證授權
說明 取得允許存取指定電腦的非管理者用戶資訊。此版本使用 session id 取代輸入用戶資訊為參數。
回傳值 允許存取指定電腦的非管理者用戶資訊。
URL POST http://<domain>:<port>/crepo/user/approved2
參數範例（無參數）
回傳值範例（格式：application/json）
[
    {
        "account" : "9af3a520656abdab5305b5b3f905b994ea10a24c16783fc340",
        "nickname" : "Charles-Émile Reynaud",
        "showHidden" : false,
        "allowAlias" : true
    },
    {
        "account" : "4ef59f26488fc451e8d73a4ef192ef659cbad89cec48b5c451b",
        "nickname" : "Louis Daguerre",
        "showHidden" : false,
        "allowAlias" : true
    }
]
回傳值說明
名稱 類型 說明
必要參數
account string 使用者帳號。
nickname string 使用者暱稱
showHidden boolean 是否顯示隱藏檔。
allowAlias boolean 是否顯示目錄與檔案連結。
非必要參數
（無）
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 該 session 未指定任何使用者或電腦編號。
401 desktop session id 已經過期。
403 session 不存在。或者該 session 所對應的用戶不存在。
收到此回傳值，前端提示用戶重新開啟應用程式，重新進行設定。
460 此電腦不存在。
補充說明
無
