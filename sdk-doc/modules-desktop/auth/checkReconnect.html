模組 認證授權
URL POST http://<domain>:<port>/crepo/ping
說明 檢查指定使用者電腦是否要求desktop建立連線
參數值說明
名稱 類型 說明
必要參數
（無）
非必要參數
（無）
回傳訊息（HTTP Status）
訊息碼 說明
200 使用者要求重新建立連線
400 未指定使用者電腦編號
401 使用者電腦編號不存在
409 已經連線，使用者並沒有要求重新建立連線
503 尚未連線，使用者也沒有要求重新建立連線
補充說明
  1. 以使用者電腦編號以 SHA-256 方式加密後的值作為此服務的http header「Authorization」值。
    系統提供此服務前，先驗證HTTP Header「Authorization」的值是否為有效的使用者電腦編號，
    若不是則回傳錯誤訊息碼401（Unauthorized）。
  2. 當回傳狀態為 200 (OK) 時，表示使用者要求與repository建立連線，否則回傳狀態為 409（Conflict），表示不需要建立與repository的連線。
  3. 若沒有找到http header「Authorization」值，回傳狀態為 400。
  4. 若發生其他錯誤，回傳狀態為 500（INTERNAL SERVER ERROR）。
