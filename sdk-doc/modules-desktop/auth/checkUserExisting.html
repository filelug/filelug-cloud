Check User Existing
回上一層
模組 認證授權
說明 確認指定使用者存在且密碼正確。
回傳值 無
URL POST http://<domain>:<port>/crepo/user/exist
參數範例（格式：application/json）
{
  "account" : "9aaa3acb2bf8aa13533040cc4b24cec22089eba94b4d1c11f3a520656abdab5305b5b3f905b994ea10a24c16783fc340",
  "passwd" : "81dc9bdb52d04dc20036dbd8313ed055",
  "nickname" : "Jellyfish",
  "verification" : "bf94b4d1c5305b5b3f96abdab89eba05b994ea10a249aaa3acb28aa13533040cc4b24cec2783fc3402011f3a52065c16",
  "locale" : "zh_TW"
}
參數值說明
名稱 類型 說明
必要參數
account string 使用者帳號。
passwd string 使用者密碼。此密碼為SHA-256加密後的密碼
nickname string 使用者暱稱
verification string 驗證碼。用來確認此服務是由使用者經由授權的device執行，而不是機器在執行。
非必要參數
（無）
回傳錯誤訊息（HTTP Status）
訊息碼 說明
400 使用者帳號與密碼至少一個未提供。
401 使用者密碼與既有的不相同。
403 使用者帳號尚未註冊。
補充說明
無
